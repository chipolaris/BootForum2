Generate Simulated Data

Use this tool to simulate User/Discussion/Vote data for testing purposes.

Generate Users

Create a specified number of simulated users with the role 'USER' and password 'fake123'.

Generate Forum Content

This will create a full set of simulated forum groups, forums, discussions, and comments. This is a one-time action to populate an empty database.

Generate Votes

This will create simulated upvotes and downvotes for all existing discussions and comments. Best used after generating forum content.
